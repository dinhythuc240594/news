Cài đặt hệ thống
Cài đặt API News
Token để truy cập API tin tức bên ngoài (ví dụ: Bearer token)
URL cơ sở của News API
Lưu ý: API Token là bắt buộc để có thể lấy tin tức từ API bên ngoài. Vui lòng đảm bảo token hợp lệ và có quyền truy cập.
Cài đặt SMTP (Email)
Địa chỉ SMTP server (ví dụ: smtp.gmail.com, smtp-mail.outlook.com)
Port SMTP (thường là 587 cho TLS hoặc 465 cho SSL)
Email đăng nhập SMTP
Với Gmail, sử dụng App Password thay vì mật khẩu thường
Email người gửi (mặc định sẽ dùng SMTP Username nếu để trống)
Chọn TLS cho port 587, SSL cho port 465
Lưu ý:
  • Với Gmail: Bật 2-Step Verification và tạo App Password tại myaccount.google.com/apppasswords
  • Với Outlook/Hotmail: Sử dụng mật khẩu tài khoản Microsoft
  • Đảm bảo SMTP Server và Port đúng với nhà cung cấp email của bạn
